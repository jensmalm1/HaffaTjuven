Haffa Tjuven är ett community med aktuella brott som tema. Användare har möjligheten att på en karta se vad för brott som skickats ut via Polisens RSS-flöde. Medlemmar kan härifrån dela med sig av observationer, spekulationer samt länkar till nyhetssidor. Övriga inlägg rensas. Haffa Tjuven ställer krav på att användare utelämnar känslig information såsom namn och övriga personuppgifter.
